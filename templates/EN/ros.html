Control your Ergo Jr with ROS

What is ROS ?

Robot Operating System (ROS) is an open-source, meta-operating system for your robot. It provides the services you would expect from an operating system, including hardware abstraction, low-level device control, implementation of commonly-used functionality, message-passing between processes, and package management. It also provides tools and libraries for obtaining, building, writing, and running code across multiple computers [http://wiki.ros.org].

How to control your Ergo Jr with ROS ?

To control your robot with ROS, you will need to run the ROS master and the poppy_controllers package which allows you to access ROS services from your Ergo Jr.
To run the ROS master and the Poppy services, simply click on the button below. Please note that you must stop the API first.
Set the variable ROS_MASTER_URI on the pc you want to control your Ergo Jr with: export ROS_MASTER_URI=http://poppy.local:11311. You can then launch Rviz or access the different services of the Poppy with rosservice (command line).
